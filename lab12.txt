каталог /var/log
/var/log/syslog или /var/log/messages - содержит глобальный системный журнал, 
в котором пишутся сообщения с момента запуска системы, от ядра Linux, 
различных служб, обнаруженных устройствах, сетевых интерфейсов и много другого.
/var/log/auth.log или /var/log/secure - информация об авторизации пользователей, 
включая удачные и неудачные попытки входа в систему, 
а также задействованные механизмы аутентификации.

Уровни приоритетов сообщений:
emerg - аварийная ситуация
alert - критическая ошибка
crit - серьезная ошибка
err - ошибка
warn - предупреждение
notice - ситуация штатная, но требует проверки
info - уведомление
debug - отладочное сообщение

пары селектор-действие
селектор=подсистема.приоритет (auth.notice)
! - приоритеты, строго меньше указанного
* - все приоритеты
= - строго указанный приоритет
для исключения подсистемы - приоритет none (подсистема.none)
/etc/rsyslog.conf - файл конфигурации rsyslogd

logrotate - архивирование старых файлов протоколов
Настройка в /etc/logrotate.conf + все файлы в /etc/logrotate.d

apt-get install logrotate
bash-скрипт для logrotate в каталоге /etc/cron.daily

logcheck - мониторинг файлов протоколов, по умолчанию просматривает файлы протоколов 
каждый час и отправляет интересный сообщения на электронную почту администратора
/etc/logcheck/logcheck.logfiles - список читаемых логов
apt-get install logcheck
/etc/logcheck/logcheck.conf - главный файл конфигурации
